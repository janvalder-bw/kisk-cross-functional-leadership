# Cross-functional Leadership

Design Management & Leadership Masterclass — presentation for KISK MUNI.

## Contents

- [**slides.md**](./slides.md) — Marp slide deck (Markdown + YAML front matter) with speaker notes
- [**slides.pdf**](./slides.pdf) — exported PDF of the presentation
- [**article.md**](./article.md) — standalone article on cross-functional leadership (Czech, for university students)

## Viewing & editing

- Edit `slides.md` in any Markdown editor.
- To preview or export, use [Marp for VS Code](https://marketplace.visualstudio.com/items?itemName=marp-team.marp-vscode) or the [Marp CLI](https://github.com/marp-team/marp-cli).

### Export with Marp CLI

```bash
# Install (requires Node.js)
npm i -g @marp-team/marp-cli

# Export to PDF
marp slides.md --pdf

# Export to HTML
marp slides.md --html
```
